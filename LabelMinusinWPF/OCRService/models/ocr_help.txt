=== OCR 功能说明 ===

【一键打点】
  弹窗选图 → PaddleOCR 检测文字区域 → 生成位置标签
  全语言适用，无需 Python，约 2-5 秒/张

【一键中英识别】
  弹窗选图 → PaddleOCR 检测+识别 → 生成文字标签
  中英文适用，无需 Python

【一键日文识别】
  弹窗选图 → PaddleOCR 检测 + manga-ocr 识别 → 生成文字标签
  日文专精，需配置 Python 环境

【截图自动识别】
  开启 OCR 开关 → 在图片上框选区域 → 自动识别并创建标签
  中英/日文可在开关旁切换引擎

【环境配置（仅日文识别需要）】
  配置环境 → 自动下载 Python + PyTorch + manga-ocr
  约 2GB，首次安装约 5-10 分钟

【模型文件】
  程序目录\models\v5\        — PaddleOCR ONNX 模型
  程序目录\models\manga-ocr\ — manga-ocr 脚本与模型

【提示】
  · OCR 结果可通过 Ctrl+Z 撤销
  · 建议先检查再保存
