=== OCR 功能说明 ===

【一键打点】
  弹窗选图 → PP-OCRv5 检测文字区域 → 在文字右上角生成位置标签
  只需要 PP-OCRv5 ONNX 模型，不需要 Python。空文本标签仍按 Label1、Label2... 命名。

【一键中英识别】
  弹窗选图 → PP-OCRv5 检测并识别文字 → 生成文字标签
  适合中文、英文和混排文本，只需要 PP-OCRv5 ONNX 模型。

【一键日文识别】
  弹窗选图 → PP-OCRv5 检测文字区域 → manga-ocr 识别日文 → 生成文字标签
  适合日漫/日文图片，需要 PP-OCRv5 ONNX 模型、Python、torch、manga-ocr 和 manga-ocr 模型。

【截图自动识别】
  选择中英或日文引擎 → 开启 OCR 开关 → 切到截图模式 → 在当前图片上框选区域
  识别结果会映射回当前图片，并创建一个标签。当前开关会预热中英和日文识别引擎。

【字体识别网站】
  先截图复制到剪贴板 → 点击“字体识别”旁的网站按钮 → 在外部网站中识别字体
  可在设置里自定义网站地址；留空时使用内置网站列表。

【环境配置】
  配置 OCR 环境 → 自动下载 PP-OCRv5 ONNX 模型、嵌入版 Python、torch、manga-ocr 和日文模型
  约 2GB，首次安装约 5-10 分钟。只做一键打点/中英批量识别时，通常只需要 PP-OCRv5 ONNX 模型。

【模型文件】
  程序目录\models\v5\        — PP-OCRv5 ONNX 检测/识别模型
  程序目录\models\manga-ocr\ — manga-ocr 脚本与日文识别模型
  程序目录\python\           — 嵌入版 Python 和 Python 包

【提示】
  · OCR 结果可通过 Ctrl+Z 撤销
  · 批量 OCR 会对所选图片逐张创建标签
  · 建议检查识别结果后再保存
